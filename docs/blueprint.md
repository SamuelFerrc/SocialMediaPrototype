# **App Name**: EchoSphere

## Core Features:

- User Authentication: Secure login/registration system for users to access the social network.
- Post Creation: Allows users to create and post messages (text-based).
- Post Liking: Enables users to 'like' posts made by others.
- Commenting System: Users can add comments to posts, including their own.
- Sentiment Analysis Tool: This tool assesses the sentiment of each comment made on a post to help moderators identify potentially harmful content.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5), a calming yet engaging shade for the app's main elements.
- Background color: Light grey (#EEEEEE), a soft and neutral background to ensure readability.
- Accent color: Warm amber (#FFB300), for interactive elements like buttons and highlights, creating a welcoming and approachable feel.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality.
- Simple, outline-style icons to represent actions and categories, maintaining a clean and modern aesthetic.
- A clean, card-based layout for posts, making it easy to browse and interact with content.
- Subtle animations for interactions, such as liking or commenting, providing gentle feedback to the user.
